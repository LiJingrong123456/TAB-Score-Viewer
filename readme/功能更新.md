# 功能更新记录

## v1.0.0 - 2026-06-06 (初始版本)

### 新增功能

#### 1. 多格式文件支持
- **图片格式**: PNG, JPG, JPEG, WEBP, BMP, GIF
- **PDF格式**: 支持多页浏览、分页导航
- **GTP格式**: GP3, GP4, GP5, GP6, GP7, GTP, PTB, TEF
- 支持点击选择和拖拽上传两种方式打开文件

#### 2. 谱面查看器
- **ImageViewer**: 
  - 鼠标滚轮缩放 (0.1x - 5.0x)
  - 左键拖动平移
  - 双击重置视图
  - 实时显示当前缩放比例
  
- **PdfViewer**:
  - 基于react-pdf(pdf.js)实现
  - 多页PDF分页显示
  - 页码导航（上一页/下一页）
  - 自适应宽度渲染
  - 加载状态动画提示

- **GtpViewer**:
  - 基于@coderline/alphatab开源库实现
  - 完整的Guitar Pro谱面渲染
  - 内置MIDI播放引擎
  - 和弦图显示
  - 曲目信息展示（标题、艺术家）

#### 3. 播放控制系统
- **PlayBar组件**:
  - 播放/暂停按钮（大号主按钮）
  - 可拖拽进度条（带手柄高亮）
  - 时间显示（当前时间/总时长 mm:ss格式）
  - 快进/快退5秒按钮
  - 循环模式切换（无循环→全部循环→区域循环）
  - 区域循环可视化高亮
  - 速度调节（0.25x - 4.0x，步进0.1x）

#### 4. 速度曲线系统
- **SpeedCurve组件**:
  - 基于Recharts的可视化曲线编辑器
  - 4种内置预设：恒定速度、渐入渐出、渐进加速、难点慢练
  - 自定义控制点添加/删除/编辑
  - 3种插值方式：线性、平滑(贝塞尔)、阶梯
  - 启用/禁用开关
  - 点击控制点进行精确编辑
  - 仅对图片/PDF格式生效

#### 5. 文本标注系统
- **AnnotationLayer组件**:
  - 覆盖在谱面上的透明标注层
  - 点击谱面任意位置添加新标注
  - 6种颜色可选（橙、蓝、绿、红、紫、粉）
  - 标注气泡样式，带三角指示器
  - 右键菜单操作（颜色切换、编辑、删除）
  - 悬停显示删除按钮
  - 单击进入编辑模式

- **NoteEditor组件**:
  - 侧边栏标注列表管理
  - 批量操作：全选删除、导入、导出
  - JSON格式导入/导出标注数据
  - 快速添加按钮
  - 显示标注位置坐标信息
  - 内联文本编辑

#### 6. UI/UX设计
- **深色主题**：护眼的暗色配色方案
- **响应式布局**：适配不同屏幕尺寸
- **流畅动画**：150-300ms过渡效果
- **自定义滚动条**：统一的深色风格
- **无障碍支持**：键盘导航、ARIA标签
- **Google Fonts**：Poppins + Righteous字体组合

#### 7. 全局快捷键
- Space: 播放/暂停切换
- Ctrl+左箭头: 减速
- Ctrl+右箭头: 加速

### 技术实现细节

- 使用Zustand进行轻量级全局状态管理
- 基于requestAnimationFrame的高性能播放定时器
- CSS类名合并使用clsx + tailwind-merge
- 组件化设计，高度模块化
- 所有代码包含详细中文注释
