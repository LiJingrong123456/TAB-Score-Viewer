# 功能更新记录

## (2026-06-20) - 最近打开文件功能 (重构后重新实现)

### 修改内容

- **最近文件列表(独立控件)**: 与文件夹文件列表完全分离的独立 QListWidget，支持折叠/展开
  - 位置: 文件夹文件列表上方，带标题行(标签+展开/折叠按钮)，两列表之间间距 8px
  - **默认折叠**: 只显示1个最新文件，点击"展开 ▼"按钮显示全部
  - 显示格式: `📄 文件名` (使用强调色/橙色，由CSS统一控制颜色)
  - 边框样式: 使用强调色边框(与普通文件列表的主色边框区分)
  - 高度自适应: 折叠时1行高度，展开时紧贴所有项数(无多余空白)
  - 单击即可快速重新打开(比普通双击更便捷)
  - 右键菜单: 打开文件 / 在资源管理器中定位 / 从列表移除

- **数据持久化**: 
  - 存储位置: `config/settings.json` 的 `recent_files` 字段
  - 存储内容: 绝对路径列表(最多4条)
  - 自动过滤不存在的文件(启动时清理)

- **自动更新逻辑**:
  - 每次通过 `show_display()` 打开文件时自动记录
  - 新文件插入列表头部(最新的在最前面)
  - 自动去重+截断(最多4条)

### 技术实现

- `_add_recent_file()`, `_refresh_recent_list()`, `_apply_recent_list_height()`, `_toggle_recent_list()`
- `_on_recent_file_clicked()`, `_show_recent_context_menu()`, `_remove_recent_file()`
- `load_config()/save_config()` 扩展 recent_files 字段
- CSS选择器: `#recent_list_widget`(强调色) vs `QListWidget`(主色)

### 修改文件
| 文件 | 改动 |
| --- | --- |
| `TAB Score Viewer.py` | SelectionWindow新增最近文件属性+8个方法+独立控件; 修改load_config/save_config/show_display等 |
| `locales/*.json` | 已有7个翻译键(recent_list_label, recent_expand, recent_collapse等) |

## (2026-06-27) - 设置面板: 添加恢复默认功能

- **新增 `_DEFAULT_RENDER_VALUES` 等默认值快照**: 在 `_RENDER_PARAMS` 定义之后捕获 `RenderConfig` 类属性的初始值（模块加载时），避免运行时修改影响恢复默认的准确性

## (2026-06-27) - 平台自适应字体 + 选择窗口重命名

### 修改内容

- **平台自适应默认字体**: 新增 `get_font_families()` / `get_font_family()` / `get_font_family_css()` 工具函数
  - 根据 `platform.system()` 自动判断 Windows / Linux / macOS
  - Windows: 优先使用 `Microsoft YaHei` / `Segoe UI` / `Consolas`
  - Linux: 优先使用 `Noto Sans CJK SC` / `WenQuanYi Micro Hei` / `DejaVu Sans` / `DejaVu Sans Mono`
  - macOS: 优先使用 `PingFang SC` / `SF Pro Text` / `Menlo`
  - 三种字体用途: `"ui"` 主界面/标注、`"numeric"` 数字/英文标签、`"mono"` 等宽数值

- **替换所有硬编码字体**: 将 `TAB Score Viewer.py` 中所有写死的 `Microsoft YaHei` / `Arial` / `Consolas` 替换为平台自适应函数
  - 样式表 `font-family` 改为 `get_font_family_css('ui')`
  - `QFont(...)` 绘制调用改为 `get_font_family('ui'/'numeric'/'mono')`
  - 标注数据模型 `Annotation.font_family` 默认值改为平台自适应 UI 字体

- **选择窗口重命名**: 将原 `SettingsWindow` 类改名为 `SelectionWindow`
  - 类定义、docstring、方法注释、程序入口实例化全部同步更新
  - 窗口标题翻译 `app.settings_title` 从「设置」改为「选择窗口」
  - 英文翻译同步改为 `Selection Window`
